# Pendeteksi-Jenis-Sampah
♻️ TUMOU-BIN Pendeteksi Sampah

Proyek ini adalah website interaktif berbasis HTML, CSS, JavaScript, dan AI yang dapat:

📷 Menggunakan kamera langsung di browser untuk mendeteksi jenis sampah (Organik, Anorganik, B3).

🤖 Memberikan penjelasan edukatif melalui Chat AI mengenai pengelolaan dan daur ulang sampah.

🌍 Meningkatkan kesadaran lingkungan dengan cara sederhana dan mudah diakses siapa saja.

🚀 Fitur Utama
Kamera live langsung dari browser.
Deteksi 3 kategori sampah:
Organik → sisa makanan, daun, ranting.
Anorganik → plastik, kaleng, kaca.
B3 (Bahan Berbahaya & Beracun) → baterai, obat, limbah medis.
Chat AI untuk bertanya seputar sampah dan daur ulang.
Tampilan website modern dan interaktif.
